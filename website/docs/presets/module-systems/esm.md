# presetter-preset-esm

Modern ES Modules configuration for cutting-edge JavaScript development.

## Purpose

This preset extends the essentials preset with ES Module specific configurations.

## Installation

```bash
npm install --save-dev presetter presetter-preset-esm
```

## Usage

```typescript
// presetter.config.ts
export { default } from 'presetter-preset-esm';
```
