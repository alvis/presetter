# Framework Presets

Framework-specific configurations that include specialized tooling and optimizations.

## Available Presets

### [@presetter/preset-web](./web)
Browser globals, DOM typings, TailwindCSS linting, and static asset typings.

### [@presetter/preset-react](./react)
React applications with TSX support and React-specific linting.

### [@presetter/preset-storybook](./storybook)
Standalone Storybook setup with accessibility checks and Vitest browser testing.
