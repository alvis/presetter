Tutorials 📚

Learn Presetter through practical, hands-on guides. Each tutorial walks you through real-world scenarios with complete examples and interactive tools.

🔧 Presetter Configuration Generator

Configure your project settings to generate the perfect Presetter setup:

📦 Installation Command

pnpm add -D presetter presetter-preset-essentials presetter-preset-esm presetter-preset-strict

⚙️ presetter.config.ts

import preset0 from 'presetter-preset-essentials';
import preset1 from 'presetter-preset-esm';
import preset2 from 'presetter-preset-strict';

export default [
  preset0,
  preset1,
  preset2
];

📋 package.json (partial)

{
  "devDependencies": {
    "presetter": "^8.0.0",
    "presetter-preset-essentials": "^8.0.0",
    "presetter-preset-esm": "^8.0.0",
    "presetter-preset-strict": "^8.0.0"
  },
  "scripts": {
    "build": "run build",
    "clean": "run clean",
    "typecheck": "run typecheck",
    "lint": "run lint",
    "format": "run format",
    "test": "run test"
  }
}

🚀 Bootstrap Command

presetter bootstrap
💡 Next Steps:
  1. Run the installation command above
  2. Create the presetter.config.ts file
  3. Run presetter bootstrap
  4. Start developing with your configured tooling!

⚡ Presetter Command Helper

Generate the right Presetter command for your needs:

Description:

Initialize Presetter configuration and generate all config files

🎯 Generated Command

pnpm exec presetter bootstrap

💡 More Examples

presetter bootstrap
presetter bootstrap --force  # Overwrite existing files
📚 Command Reference:
  • bootstrap: Set up initial configuration files
  • run: Execute a single script with optional arguments
  • run-s: Run scripts one after another (sequential)
  • run-p: Run scripts at the same time (parallel)
